# __Тестовое задание__

#### По заданию было необходимо написать скрипт, который проходит опрос по следующему адресу http://185.204.3.165

> __Перед началом необходимо склонировать репозиторий следующим образом :__

```git clone https://github.com/yakomisar/crawler.git```

> __Или так:__

```git clone https://github.com/yakomisar/crawler.git Crawler```
- - - -

### __Запуск проекта__
После клонирования репозитория есть два вариант запуска скрипта

> __Вариант №1__

1. Добавляем/обновляем все недостающие модули:

```go mod tidy```

2. Добавляем/обновляем все недостающие модули:

```go run .```

> __Вариант №2 (MAC OS)__

В командной строке запускаем скрипт командой ```./crawler```
